Clase que maneja la configuración global de la aplicación.

Esta clase utiliza el patrón singleton para asegurar que sólo exista una instancia de configuración. Se definen parámetros tales como el modo de creación del sistema, el número de objetos, el tiempo de simulación, el archivo de datos a cargar y los algoritmos utilizados para la detección de colisiones y la resolución.

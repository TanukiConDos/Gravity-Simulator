Descripción detallada

Clase encargada de construir y gestionar el pipeline gráfico de Vulkan.

Esta clase se encarga de crear el pipeline gráfico utilizando la GPU y un render pass proporcionado. Además, configura el layout de descriptor sets y el pipeline layout.

Descripción detallada

Sistema físico encargado de actualizar la simulación mediante la detección de colisiones y la resolución de fuerzas.

La clase utiliza una instancia de CollisionDetectionInterface para detectar colisiones entre objetos físicos y una instancia de SolverInterface para resolver las interacciones y actualizar el estado de dichos objetos.

Descripción detallada

Encapsula un pool de comandos de Vulkan y gestiona múltiples buffers de comando.

La clase utiliza una referencia a GPU para crear el VkCommandPool, y ofrece métodos para administrar la creación, inicio, finalización y reinicialización de VkCommandBuffer.

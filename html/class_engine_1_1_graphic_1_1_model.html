Descripción detallada

Representa un modelo 3D generado a partir de parámetros geométricos.

La clase Model se utiliza para generar la malla del modelo, almacenar los vértices e índices, y crear el buffer correspondiente en la GPU para su renderización.

Constructor de Model.

Genera la malla del modelo utilizando el número de sectores y pilas indicados, e inicializa los recursos necesarios en la GPU mediante la GPU y el CommandPool proporcionados.

Parámetros
sectorCountNúmero de divisiones alrededor del eje (sectores).
stackCountNúmero de divisiones en la dirección vertical (pilas).
gpuReferencia a la GPU para crear los buffers.
commandPoolPool de comandos utilizado para la creación de comandos de copia.
